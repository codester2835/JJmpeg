# JJmpeg
>The code provided here can be opened with Visual Studio 2026.
## Usage
> To use, clone the repository and open JJmpeg.slnx with Visual Studio 2026.  One copy of the complete tool is included with the download, feel free to edit.

## Cloning
To clone:
* Run in Command Prompt
```
git clone -b dev --single-branch https://github.com/codester2835/JJmpeg.git
```
* In File Explorer, open the folder JJmpeg\ and open JJmpeg.slnx
